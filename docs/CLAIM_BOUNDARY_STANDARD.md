# Claim Boundary Standard

Every output should be compiled into one explicit claim type:

```text
observed_fact
source_backed_claim
computed_result
symbolic_simulation
prediction
proposal
governance_marker
blocked_claim
unknown
```

Blocked claims:

```text
legal_compliance_claim
security_certified_claim
production_customer_ready_claim
medical_claim
neurological_or_cognitive_claim
diagnostic_or_treatment_claim
financial_operation_claim
autonomous_real_world_action_claim
AGI_claim
ASI_claim
```

Rule: a module may succeed while the claim remains blocked.
